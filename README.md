# AI Fireworks Comparison

A side-by-side comparison of fireworks animations generated by different AI models (Claude Code: Haiku 4.5, Opus 4.5, Sonnet 4.5 | Antigravity: Gemini Flash, Gemini Pro, Claude models).

Open `index.html` in your browser to view all animations with customizable layout options (rows, model selection, name labels) - fully responsive for desktop, tablet, and mobile.
